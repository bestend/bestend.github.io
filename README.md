# Bestend's Blog

Personal blog and portfolio website built with Hugo.

🌐 **Visit:** [https://bestend.github.io/](https://bestend.github.io/)

## About

This site showcases my work as an AI/ML Engineer and features blog posts about open source projects and software development.

## Tech Stack

- **Static Site Generator:** [Hugo](https://gohugo.io/)
- **Theme:** [Yinyang](https://github.com/joway/hugo-theme-yinyang)
- **Hosting:** GitHub Pages
- **Languages:** English & Korean (multilingual support)

## Local Development

```bash
# Install Hugo
brew install hugo

# Run local server
hugo server -D

# Build site
hugo
```
