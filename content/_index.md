## 최근 프로젝트

### 🔧 **Confee** - Configuration Management Library

Confee는 Python 애플리케이션을 위한 간단하고 타입 안전한 설정 관리 라이브러리입니다.

- **Hydra 스타일의 설정 관리** + **Pydantic 타입 안전성** + **Typer 스타일의 자동 도움말 생성**
- YAML/JSON 자동 감지
- CLI 인자, 환경 변수, 설정 파일의 유연한 오버라이드
- 중첩 필드 접근 (닷 표기법)
- 엄격한 모드/비엄격한 모드
